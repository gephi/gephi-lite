## Buttons

- Two types: default and toggle
- Can contain an optional leading icon
- Three styles: filled, outlined, and text
- One size : medium
- Keep labels concise and use sentence case

### Guidelines

Too many buttons on a screen can disrupt the visual hierarchy. Consider placing additional actions in a navigation rail, set of chips, text links, or icon buttons.
